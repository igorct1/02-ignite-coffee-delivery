# Descrição
Terceiro projeto do Ignite, uma cafeteria fictícia.


## Tecnologias

-   React.js (vite)
-   TypeScript
-   React hook form
-   zod
-   Immer
-   Styled Components



## Como utilizar?

**Clone o repositório**

```bash
$ git clone https://github.com/igorct1/02-ignite-coffee-delivery.git
```

**Instale as dependências**

```bash
$ npm install
```

**Rode o projeto**

```bash
$ npm run dev
```
